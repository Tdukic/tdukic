Osobna web stranica

Toni Dukić

Moje ime je Toni Dukić. Rođen sam 1994. godine u Splitu, gdje trenutno i živim. Ovo je moja prva web stranica napravljena u HTML i CSS. HTML i CSS sam počeo učiti prije 7 dana premda sam žalju za poznavanjem programiranja imao puno ranije. želim naučiti i druge programske jezike i s njima u budućnosti razvijati vlastite webstranice i softvere. Za sada razgledajte malo moju basic web stranicu gdje sam uklopio dio onog što sam naučio.
